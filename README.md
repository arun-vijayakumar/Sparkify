# Sparkify
Udacity Nanodegree Capstone Project
## Objective:
### Analyze the Saprkify customer log file to identify the potential customer churn
Medium blog about this project can be viewed <a href="https://medium.com/@arunvkumar_85287/sparkify-62b568cbad68"><b>here</b></a>
### Installation
This project is developed in <b>Python</b> using the <b>Jupyter</b> IDE. </br>
### Project Motivation
Spark is currently one of the most popular tools for big data analytics. This project provide an insight to the Big Data analysis with Spark.

### Results
The following 4 machine learning algorithms are used and built the customer churn predicton model.
1. Logistic Regression
2. Gradient Boosted Trees
3. Support Vector Machine
4. Random Forest
